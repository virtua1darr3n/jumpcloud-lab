# Lab Objectives

## Purpose
Build a real-world IT & InfoSec lab using JumpCloud.

## Goals
- Understand identity and access management
- Simulate onboarding and offboarding
- Prepare for device management
- Document everything clearly in GitHub

## Design Approach
This lab will simulate a small company environment using:
- JumpCloud for identity and device management
- Multiple users for lifecycle testing
- GitHub for structured documentation
