# JumpCloud + Google Workspace Lab

## Overview
This lab is being built to gain hands-on experience with identity management, device management, security controls, and SaaS integration.

The aim is to simulate a small real-world IT and InfoSec environment using JumpCloud as the central identity and device-management platform, with Google Workspace added later as a core SaaS service.

## Objectives
- Understand how JumpCloud manages users and devices
- Simulate onboarding and offboarding processes
- Apply security controls such as MFA and policies
- Explore SaaS integration using Google Workspace
- Document the lab clearly in GitHub

## Planned Architecture
Admin → JumpCloud → Devices & SaaS Applications
