# JumpCloud Lab – Cloud Identity & Device Management

## Overview
This lab was created to gain hands on experience with JumpCloud as a cloud based directory service.

The goal was to understand how modern identity platforms manage users, devices, and security policies.

## Lab Environment
- Windows 11 (primary device)
- JumpCloud Free Tier
- Test user account

## What I Configured
- User creation and management
- Device enrolment using JumpCloud agent
- User-to-device binding
- Authentication testing (JumpCloud login)
- Multi-Factor Authentication (MFA)
- Basic security policies (password + screen lock)

## Learning Points
- How cloud identity replaces traditional Active Directory
- Device management across platforms
- Importance of MFA in securing user accounts
- Centralised policy enforcement
